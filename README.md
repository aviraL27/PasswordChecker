# Password Strength Checker

A simple frontend project that checks the strength of a password based on common security rules.

## Features
- Minimum length validation
- Uppercase & lowercase letter check
- Digit and special character check
- Clear feedback for missing requirements
- Clean UI with instant visual feedback

## Tech Stack
- HTML5
- CSS3
- JavaScript (DOM & Events)

## Purpose
Built to practice JavaScript events, conditional logic, and basic DOM manipulation.
